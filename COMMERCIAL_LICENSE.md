# Commercial License Framework (Draft v0.1)

This document describes the intended commercial licensing model for `mx20022-runtime`.

Status: Draft framework for evaluator and procurement discussions. This is not legal advice and not a signed contract.

## 1. Open source baseline

`mx20022-runtime` is available under `AGPL-3.0-only`.

Organizations that can comply with AGPL obligations may use the runtime under AGPL.

## 2. Commercial license option

A separate commercial license is available for organizations that need to:

- deploy or embed `mx20022-runtime` without AGPL copyleft obligations,
- avoid source-disclosure requirements triggered by AGPL network use,
- negotiate enterprise terms (support, indemnity, warranties, audit rights, SLAs).

## 3. Scope of commercial grant (intended)

A commercial agreement is expected to include:

- non-exclusive rights to use, modify, and distribute binaries internally;
- production deployment rights across agreed environments/regions;
- rights to maintain private forks and internal patches;
- negotiated terms for redistribution/embedded use (if required).

## 4. Enterprise terms (intended)

The enterprise package may include:

- support and response-time commitments,
- upgrade and patch access,
- security advisory handling,
- contractual warranty and liability terms,
- optional training and implementation support.

## 5. Exclusions and dependencies

This commercial model covers `mx20022-runtime` code in this repository.

`mx20022-model`, `mx20022-parse`, and `mx20022-validate` are separate Apache-2.0 licensed upstream libraries and are not relicensed by this document.
